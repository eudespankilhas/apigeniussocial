API GENIUS SOCIAL IA

Bem-vindo! Esta é uma API baseada em Serverless Functions na Vercel, com integração Stripe.

Endpoints

  • POST /api/checkout – cria uma Checkout Session
  • POST /api/create-payment-intent – cria um Payment Intent
  • POST /api/stripe-webhook – webhook do Stripe (corpo bruto)
  • POST /api/confirm-subscription – confirmar assinatura (exemplo)

Como testar

  1. Configure variáveis na Vercel: STRIPE_SECRET_KEY, APP_URL, e depois crie o webhook e defina STRIPE_WEBHOOK_SECRET.
  2. Faça um POST para /api/checkout com priceId ou amount.
  3. Finalize o pagamento e verifique eventos no painel da Stripe (Payments/Events).

Dica: use Invoke-RestMethod (PowerShell) ou curl para testar.
